# agenda-contatos-c
Sistema de gerenciamento de contatos em modo console,
desenvolvido em C ANSI.

## Funcionalidades
- Incluir, listar, consultar e excluir contatos
- Busca parcial pelo nome
- Dados salvos em arquivo — persistem entre execuções

## Conceitos utilizados
- Structs, vetores e funções
- Manipulação de strings
- Leitura e gravação em arquivo binário
